# Memory Game

A modern and responsive Memory Game built with React and TypeScript. Challenge your memory by finding all pairs in the shortest time and with the fewest moves!

## 🚀 Live Demo

Check out the live version here: [https://YujiSeto.github.io/memory-game](https://YujiSeto.github.io/memory-game)

## ✨ Features

- **Real-time Tracking:** Track your elapsed time and total moves.
- **Responsive Design:** Playable on desktop and mobile devices.
- **Dynamic Grid:** Cards are shuffled randomly every time you start or reset.
- **Visual Feedback:** Smooth transitions and clear states for flipped, matched, and hidden cards.

## 🛠️ Technologies Used

- **React:** For building the user interface.
- **TypeScript:** For type safety and better developer experience.
- **Styled Components:** For modern, component-based styling.

## 🏃 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YujiSeto/memory-game.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd memory-game
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm start
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
